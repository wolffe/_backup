1. DESCRIPTION

Butterfly Multi Estate Manager is a full option realty CMS solution for realtors, individuals and webmasters, a browser-based (PHP/MySQL) solution for multiuser realty listing.

Multi Estate Manager features an administration section accessible only for admin user. Upon installation, the default user is administrator (admin). Approve, edit or delete user-added listings, add or remove unlimited categories and change basic site options, such as title and currency.

Detail listing features include lightbox effect for gallery images, Google Maps and feature icons.

The system allows both free listing and PayPal IPN payment. Switch the payment type (free or not) from the Administration | Settings screen.

Compatibility

Butterfly Multi Estate Manager has been tested in Microsoft Internet Explorer, Mozilla Firefox, Google Chrome and Safari and works fine.

2. REQUIREMENTS

Server requirements for Butterfly Multi Estate Manager are:

  • PHP 5+
  • MySQL 5+

Butterfly Multi Estate Manager has been built on the following configuration:

  • Apache/2.0.58 (Win32)
  • PHP/5.2.3
  • MySQL Server 5.0.41-community-nt
  • MySQL Server 5.0.22

3. INSTALLATION

Check your includes/config.php file for correct database settings. You can find a clean SQL dump in /install/ directory. Import it in your database. You're done!

The default username and password are, as noted above, admin/1234. The admin user has access to administration section.

4. UPGRADE

Butterfly Multi Estate Manager gets updated by overwriting the files.

Moreover, please check the install folder, for a special update file, depending on your Estate Manager version.
