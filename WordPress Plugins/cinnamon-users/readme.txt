== Description ==

Cinnamon Users allows you to create slick looking user profiles for multi-author WordPress sites. Cinnamon Users features lots of settings and options and allows for unique looking author portfolio pages.

== Installation ==

1. Upload the 'cinnamon-users' folder to your '/wp-content/plugins/' directory
2. Activate the plugin via the Plugins menu in WordPress
4. A new Cinnamon Users will appear in WordPress with options and general help
