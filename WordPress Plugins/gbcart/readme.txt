== Description ==
GBCart is a mail order shopping cart. No payment gateways are available, instead the order goes straight to administrator's email.

== Installation ==

1. Upload the wp-ecards folder to your '/wp-content/plugins/' directory
2. Activate the plugin via the Plugins menu in WordPress
3. Create and publish a new page/post
4. Add one of the available shortcodes
