== Description ==

WordPress Placeholder is a quick and simple function for inserting image placeholders in your design or code. Add the shortcode in your post or page (or straight into your template) and get a customized placeholder.

== Installation ==

1. Upload the wp-placeholder folder to your '/wp-content/plugins/' directory
2. Activate the plugin via the Plugins menu in WordPress
3. A new Placeholder menu will appear in WordPress with options, and general help and tips
